# Key Binds

> ⚠️ **This feature is currently under development and is not available in the public build.**

---

## Overview

Key binds will allow you to map a spoken phrase to a keypress, letting you trigger in-game actions by voice. For example, saying a phrase could press a specific key in your game without you touching the keyboard.

This feature is being reworked for reliability across different games and input systems. Check back in a future update.

---

## Planned Usage

Once available, key binds will be configured in the **Actions** tab of the IPA UI. You will be able to:

- Set a phrase to trigger any key or key combination
- Set a repeat count for keys that need to be pressed multiple times
- Test the bind before saving

> **Note:** Key binds will not work in games protected by anti-cheat software such as EasyAntiCheat or BattlEye. They are intended for games that allow external input or for use outside of games entirely.
